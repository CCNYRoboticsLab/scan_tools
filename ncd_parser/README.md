DESCRIPTION:
-----------------------------------

The ncd_parser package reads in .alog data files from the New College Dataset and 
broadcasts scan and odometry messages to ROS.

INSTRUCTIONS:
-----------------------------------

To compile, see scan_tools's `README.md`

To test, run:

    roslaunch ncd_parser demo.launch
